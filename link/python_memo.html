OpenCV関連

参考： SIFT, SURFが利用できるPython用OpenCVをインストールする

2種類あるよ

  • opencv-python : メインモジュールのみ
  • opencv-contlib-python : メインモジュール＋contlibモジュール（SIFT, SURF, CUDA関連などを使う場合必要）
各モジュールは以下で確認可能．
https://docs.opencv.org/master/

pipでインストール

$ pip install opencv-contrib-python
$ pip install opencv-python

バージョンやモジュールをpython上で確認

import cv2
cv2.__version__ # バージョン確認
dir(cv2.ml) # mlモジュールの確認
